# ⚡ PWR Chain Validator Node 

Welcome to the PWR Chain inaugural testnet! This guide helps you set up a validator node

---

## 📋 Requirements

- **CPU:** 2 vCPU  
- **RAM:** 4 GB  
- **Disk:** 100 GB or higher  
- **Bandwidth:** 400Mbps (down), 800Mbps (up)  
- **Open Ports:**
  - TCP: 8231, 8085, 9864
  - UDP: 7621
